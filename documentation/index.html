Class Index

  • soma

    Global namespace and contains some helpers.

  • soma.core

    Contains the SomaCore class Application (entry point of the framework).

  • soma.core.Application

    Introduction
    SomaCore is a lightweight event-based MVC framework written in javascript that provides a structure, models, views management and commands.

  • soma.core.controller

    Contains classes related to commands and events.

  • soma.core.controller.Command

    Class that will be instantiated when a registered event is dispatched, the framework will automatically call the execute method.

  • soma.core.controller.ParallelCommand

    The ParallelCommand class is used to execute a list of commands, they will all start at the same time.

  • soma.core.controller.SequenceCommand

    The SequenceCommand class is used to execute a list of commands one after the other.

  • soma.core.controller.SomaController

    The SomaController handles the commands added to the framework and the events dispatched from either a display list or a framework element (instance of the framework, commands or wires).

  • soma.core.IDisposable

    This interface provides the method that can be called to dispose the elements created inside this instance.

  • soma.core.IResponder

    This interface provides the contract for any service that needs to respond to remote or asynchronous calls.

  • soma.core.mediator

    Contains a Mediator abstract class.

  • soma.core.model

    Contains the models manager and a Model abstract class.

  • soma.core.model.Model

    The model is the class used to manage you application's data model.

  • soma.core.model.SomaModels

    The SomaModels class handles the models of the application.

  • soma.core.view

    Contains the views manager and a View abstract class (no framework dependency).

  • soma.core.view.SomaViews

    The SomaViews class handles the views of the application.

  • soma.core.wire

    Contains the wires manager and a Wire abstract class.

  • soma.core.wire.SomaWires

    The SomaWires class handles the wires of the application.

  • soma.core.wire.Wire

    A Wire is a class that will hold the logic of the Application.

  • soma.Event

    Event wrapper class for the native event created with "document.createEvent".

  • soma.EventDispatcher

    Class on which on you can add and remove listeners of an event.

  • soma.View

    The View class is not dependant of the framework and is completely optional.
